1. Для ввода комплексного числа используйте кнопки (EXP) и (+j).
   1.1 Для ввода мнимой части комплексного числа нажмите (+j).
   1.1 Для ввода аргумента экспоненты  нажмите  (EXP).

2. Для перевода комплексного числа из алгебраической формы в  показательную нажмите кнопку (EXP/ALG).

3. Для извлечения квадратного корня действительного числа нажмите кнопку (SQRT).

4. Для запоминания текущего значения нажмите кнопку (MEMORY+), а для вывода  из памяти - кнопку (MEMORY-).

5. Для удаления текущего числа или оператора нажмите кнопку (BACK).

6. Для сброса всех чисел и операторов кроме числа в памяти нажмите  кнопку (C).

7. Результат вычислений по умолчанию представляется в алгебраической форме.

8. Над результатом вычислений можно продолжать выполнять операции.
